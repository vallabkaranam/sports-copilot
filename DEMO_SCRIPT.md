# Demo Script

## Goal
Record one clean 2-minute single take that proves AndOne is a Track 3 sidekick:
- the human stays in control
- the system stays quiet while confidence is high
- the system intervenes only when hesitation appears
- the cue is grounded and transparent
- the cue weans off once the speaker recovers

## Roles
- `Teammate A`: live commentator
- `Teammate B`: narrator / operator

## Screen Focus
Keep the recording focused on:
- replay/video area
- hesitation/confidence meter
- cue card
- cue fade / backing off
- `Why this cue`

Do not spend time showing setup, settings, or deep analytics in the main take.

## Exact Recording Flow
1. Start on the live booth with the replay loaded and ready.
2. Click `Go live`.
3. Press `Play`.
4. Let `Teammate A` deliver one strong line with no hesitation.
5. At the save moment, trigger hesitation with an unfinished phrase.
6. Let the cue appear.
7. Let `Teammate A` recover naturally from that cue.
8. Leave a beat so judges can see the cue shrink / fade.
9. Trigger a second hesitation using `um...` and a repeated start.
10. Let the second cue appear.
11. Click `Why this cue` for 2-3 seconds.
12. Close with the Track 3 positioning line.

## Backup Plan
- If the first hesitation window is missed, trigger `Force Hesitation` and continue.
- If the cue appears early, frame it as anticipation: `it steps in before the moment fully collapses.`
- If `Why this cue` feels risky, skip it and use the time for a cleaner ending.
- If only one hesitation lands cleanly, keep that one and shorten the ending.
