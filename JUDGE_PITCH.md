# Judge Pitch

Sports Copilot is an AI sidekick for live sports commentary, not an auto-commentator.

## The Problem
Live commentators do not need full replacement. They need help exactly when the moment gets noisy, fast, or awkward:
- a big chance
- a save
- a momentum swing
- a hesitation or dead-air pocket

## What Makes This Different
Sports Copilot fuses multiple streams into one evolving world model:
- replay event feed
- commentator transcript
- co-host state
- narrative memory
- retrieval context
- social pulse
- sampled visual cues

Then it does something subtle: it decides whether the human actually needs help. If yes, it shows one compact grounded assist with source chips. If not, it stays quiet.

## What Judges Should Notice
- The UI is broadcaster-facing, not chat-facing.
- The system suppresses itself while the co-host is still active.
- The assist is short, timely, and supported.
- The style mode can flip the assist behavior on the same replay moment.
- The entire demo path works from local deterministic fixtures, so it is reliable under demo conditions.

## Why It Matters
This is the shape of a real AI sidekick product:
- assistive, not replacing the operator
- aware of context across messy streams
- grounded instead of hallucination-prone
- capable of becoming live and productionized later without changing the core agent design
